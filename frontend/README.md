# AI Mock Interviews - UI

The frontend for the AI Mock Interviews platform, built with **React**, **Vite**, and **Framer Motion**.

## 🚀 Quick Start

### 1. Prerequisites
Ensure you have [Bun](https://bun.sh/) (recommended) or [Node.js](https://nodejs.org/) installed.

### 2. Installation
Clone the repository and install dependencies:
```bash
bun install
# or
npm install
```

### 3. Development
Start the development server:
```bash
bun run dev
# or
npm run dev
```
The application will be available at [http://localhost:5173](http://localhost:5173).

---

## 🔗 Backend Connection
This frontend requires the [AI Mock Interviews Backend](https://github.com/roman/ai-mock-interviews) to be running. By default, it looks for the API at `http://localhost:8000`.

To start the backend (using Docker):
```bash
cd ../ai-mock-interviews
docker compose up -d
```

---

## 🛠 Tech Stack
- **React 19** - UI Library
- **Vite** - Build Tool
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **Axios** - API Requests
- **React Router** - Navigation
